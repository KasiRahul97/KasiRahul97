## ⭐ Featured Projects

### 1️⃣ **CloudSense** — Leakage-Free Cloud CPU Forecasting
**Python · PyTorch · FastAPI · Docker · AWS**

[→ View Repository](https://github.com/KasiRahul97/CloudSense)

Causal CPU forecasting benchmark that lives behind a live FastAPI + Docker stack. Online predictions reproduce offline test accuracy exactly—no train/serve skew.

**Key Wins:**
- **R² 0.73 / MAE 5.7%** at 4-hour-ahead horizon (CEEMDAN + CNN-BiLSTM ensemble)
- Found & fixed a **data-leakage bug** that had tanked a baseline to **R² ≈ −1.7** → recovered to **≈ 0.70**
- Tested 7 models end-to-end (deep learning + naive baselines) with honest comparisons
- 30 passing tests, GitHub Actions CI, live demo dashboard

**Why it matters:** Proves forecasting accuracy in production, not just Jupyter notebooks.

### 2️⃣ **PAXA AI Assistant** — Multi-Agent AI Orchestration
**Python · LLM · Memory Systems**

[→ View Repository](https://github.com/KasiRahul97/PAXA-AI-Assistant)

Modular AI assistant with multi-agent architecture, separating orchestration from execution. Built persistent memory so the assistant retains context across sessions.

**Architecture:**
- `core/` — orchestration layer coordinating specialized agents
- `agents/` — autonomous execution units
- `services/` — backend logic
- `memory/` — persistent state (paxa_memory.db, paxa_enterprise.db)
- `frontend/` — user interaction layer

**Why it matters:** Shows full-stack AI system design, from agent design to deployment.
