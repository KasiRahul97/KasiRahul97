### 1️⃣ CloudSense — Leakage-Free Cloud CPU Forecasting
**Python · PyTorch · FastAPI · Docker · AWS**

A reproducible, causal benchmark of CPU-utilization forecasters for proactive autoscaling, served live behind a FastAPI + Docker stack whose online predictions reproduce offline test accuracy exactly.

- **R² 0.730 / MAE 5.72%** at the 4-hour-ahead horizon (CEEMDAN + CNN-BiLSTM ensemble) on real Numenta NAB data
- Found & fixed a **causality leak**: RevIN normalization took the LSTM baseline from **R² ≈ −1.70 → ≈ 0.70**
- Benchmarked **7 models** — deep learning + naive baselines — under one honest, leakage-free protocol
- 30 passing tests, GitHub Actions CI, Dockerized inference API + live dashboard

<details>
<summary><b>Why it's worth a look</b></summary>
<br/>

The headline isn't a state-of-the-art claim — it's that under a fair, causal evaluation, naive persistence is brutally hard to beat, and the project says so out loud instead of hiding it. That kind of rigor (causal decomposition, RevIN for regime shift, real train/serve parity) is the actual engineering story.

</details>
